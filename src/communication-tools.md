# 远程团队沟通工具

## 为什么不是微信、QQ？

微信和 QQ 是我们在绝大多数场景下使用的聊天工具。但是，这二者都有很多问题，比如，无法长久保留文件、无法方便的回溯记录等。此外，工作和生活使用相同的工具，也会让我们渐渐模糊工作和生活的界限。因此，我推荐远程工作团队使用专用的沟通工具来完成沟通。同时，借助其提供的第三方拓展，优化自己的团队工作效率。
